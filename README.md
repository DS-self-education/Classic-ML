# Classic-ML
Лабораторные работы по курсу "Машинное обучение"

- **Лабораторная 1. Прогнозирование стоимости недвижимости**  
  Очистка и анализ данных о квартирах, обработка выбросов и пропусков, построение регрессионных моделей (в т.ч. ансамблей) для прогнозирования цены и анализ наиболее важных факторов, влияющих на стоимость.

- **Лабораторная 2. Прогнозирование оттока клиентов**  
  Построение модели классификации для задачи оттока клиентов: предобработка данных, борьба с дисбалансом классов (SMOTE), обучение моделей (Random Forest и др.), оценка качества по F1-мере и формулировка бизнес-рекомендаций по удержанию.

- **Лабораторная 3. Кластеризация стран по уровню развития**  
  Использование методов кластеризации (K-means и др.) для разбиения стран на группы по социально-экономическим показателям, выявление стран с наибольшей нуждаемостью в помощи и интерпретация получившихся кластеров.

- **Лабораторная 4. Кредитный скоринг и ансамблевые методы**  
  Моделирование вероятности серьёзной просрочки по кредиту (Delinquent90) на основе признаков клиента с применением ансамблей (бэггинг, стекинг): анализ данных, обучение нескольких моделей и сравнение их качества.

- **Лабораторная 5. Прогнозирование временного ряда**  
  Анализ временного ряда занятости (employment): исследование тренда и сезонности, приведение ряда к стационарному виду, подбор и обучение SARIMA-модели, оценка качества прогноза и анализ остатков.
